Welcome to UI/UX 101

Learn the fundamentals of user interface and user experience design. No experience needed — just curiosity and creativity!

What You’ll Learn

In this course, we’ll explore the basics of how people interact with digital products. You’ll learn how to design with empathy, create wireframes, and understand visual hierarchy.

Course Topics

  • Understanding users and their needs
  • Design principles: color, typography, and layout
  • Wireframing and prototyping
  • Usability testing and feedback

About This Class

This course is designed for beginners — no prior experience required! You’ll gain hands-on experience and build your first UI project by the end of the semester.
